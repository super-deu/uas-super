Administrasi Server · Cloud Computing II

Web Statis & Web Dinamis
siap di-deploy ke AWS

Halaman statis ini adalah salah satu dari dua aplikasi yang di-orkestrasi dengan Docker Compose dan dirilis otomatis melalui CI/CD GitHub Actions.

Selengkapnya

Tentang Proyek

Proyek UAS ini mendemonstrasikan Zero-Touch Deployment: perubahan kode di komputer lokal cukup di-git push, lalu pipeline membangun image, mengirimkannya ke server, dan me-restart kontainer di AWS EC2 tanpa downtime yang berarti.

Arsitektur Singkat

Web Statis

HTML & CSS murni, disajikan via reverse proxy pada port 80.

Web Dinamis

PHP + MariaDB dengan panel admin & database ter-seeding otomatis.

Orkestrasi

Docker Compose: jaringan internal, volume persisten, dan variabel environment.

CI/CD

GitHub Actions: build → push image → deploy otomatis ke AWS EC2.
